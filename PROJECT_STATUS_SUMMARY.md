## 🎓 HOW TO USE

### For Development
1. **Install Dependencies**: `npm install`
2. **Build CSS**: `npm run build:css`
3. **Watch Mode**: `npm run watch:css`
4. **Generate Sitemap**: `npm run generate:sitemap`

### For Testing
1. Open `gallery.html` in browser
2. Follow `GALLERY_TESTING_GUIDE.md`
3. Test on multiple devices
4. Check browser console for errors

### For Deployment
1. Build optimized CSS: `npm run build:css`
2. Test all pages
3. Upload to web server
4. Verify sitemap.xml is accessible
